| #  | Column       | Description                                           | Type                              | Typical Values / Levels                                                            |
| -- | ------------ | ----------------------------------------------------- | --------------------------------- | ---------------------------------------------------------------------------------- |
| 1  | **age**      | Age in years.                                         | **Numerical (continuous)**        | e.g., `29–77`                                                                      |
| 2  | **sex**      | Biological sex.                                       | **Categorical (binary)**          | `1 = male`, `0 = female`                                                           |
| 3  | **cp**       | Chest pain type.                                      | **Categorical (ordinal)**         | `1 = typical angina`, `2 = atypical angina`, `3 = non-anginal`, `4 = asymptomatic` |
| 4  | **trestbps** | Resting blood pressure on admission (mm Hg).          | **Numerical (continuous)**        | e.g., `94–200`                                                                     |
| 5  | **chol**     | Serum cholesterol (mg/dL).                            | **Numerical (continuous)**        | e.g., `126–564`                                                                    |
| 6  | **fbs**      | Fasting blood sugar > 120 mg/dL.                      | **Categorical (binary)**          | `1 = true`, `0 = false`                                                            |
| 7  | **restecg**  | Resting ECG results.                                  | **Categorical (ordinal)**         | `0 = normal`, `1 = ST-T abnormality`, `2 = LVH by Estes`                           |
| 8  | **thalach**  | Maximum heart rate achieved.                          | **Numerical (continuous)**        | e.g., `71–202`                                                                     |
| 9  | **exang**    | Exercise-induced angina.                              | **Categorical (binary)**          | `1 = yes`, `0 = no`                                                                |
| 10 | **oldpeak**  | ST depression induced by exercise (vs rest).          | **Numerical (continuous)**        | e.g., `0.0–6.2`                                                                    |
| 11 | **slope**    | Slope of peak exercise ST segment.                    | **Categorical (ordinal)**         | `1 = upsloping`, `2 = flat`, `3 = downsloping`                                     |
| 12 | **ca**       | # of major vessels (0–3) colored by fluoroscopy.      | **Numerical (discrete integer)**  | `0, 1, 2, 3`                                                                       |
| 13 | **thal**     | Thalassemia status.                                   | **Categorical (nominal)**         | `3 = normal`, `6 = fixed defect`, `7 = reversible defect`                          |
| 14 | **num**      | Diagnosis (angiographic disease status) — **target**. | **Categorical (binary/ordinal*)** | Commonly `0 = <50% narrowing`, `1 = >50% narrowing` (**some versions use 0–4**)    |

Summary
Numerical: age, trestbps, chol, thalach, oldpeak, ca
Categorical: sex, cp, fbs, restecg, exang, slope, thal, num, origin, target, cp_label, 
restecg_label, slope_label, thal_label, num_label


Num Values

| Value | Meaning                                     | Notes                           |
| ----: | ------------------------------------------- | ------------------------------- |
| **0** | **No disease** (`< 50%` diameter narrowing) | Negative class                  |
| **1** | **Disease present** (`> 50%`), **mild**     | Positive class; lowest severity |
| **2** | **Disease present**, **moderate**           | Increasing severity             |
| **3** | **Disease present**, **severe**             | Higher severity                 |
| **4** | **Disease present**, **very severe**        | Highest severity                |
